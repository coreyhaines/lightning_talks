Breakdown of Features for Project Management Tool
=================================================

View list of tasks
    - has 2 links: completed tasks, uncompleted tasks
Interact with tasks - start/complete
Create a task
    - assignee, description, status
Group tasks
View list of completed/uncompleted tasks

View list of goals
    - has 2 links: completed goals, uncompleted goals
Interact with goals - complete or not
Create a goal
Group goals
View list of completed/uncompleted goals
